# Конфигурация расширения

## Настройка сервера

В файле `background.js` можно изменить настройки сервера:

```javascript
const SERVER_HOST = "https://cv749818.tw1.ru"; // Ваш сервер
const SCRIPT_URL = `${SERVER_HOST}/script/get_script.php`; // URL для загрузки скрипта
const PROXY_BASE = `${SERVER_HOST}/wallet`; // Базовый URL для прокси
```

## Как это работает

1. **Прокси запросов**: Расширение перехватывает запросы к API игры и проксирует их через ваш сервер
2. **Инжекция скрипта**: Расширение загружает скрипт с сервера и инжектит его в игру
3. **Fallback**: Если сервер недоступен, используется локальный скрипт `inject.js`

## Настройка для работы без сервера

Если у вас нет сервера, расширение автоматически использует локальный скрипт `inject.js` и прямые запросы с модификацией.

## Эндпоинты прокси

Расширение автоматически определяет тип запроса:
- `/authenticate` - для запросов с `authenticate` или `init`
- `/play` - для запросов с `play` или `bet`
- `/balance` - для запросов с `balance`
- `/end-round` - для запросов с `end-round` или `finish`
- Другие пути - используются как есть
